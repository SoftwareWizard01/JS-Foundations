Pure V/S Impure functions

Pure
-->Aisa function jo ki bahar ke value ko naa badle wo hai pure function.
E.g.:
let a = 12;

function(){
    console.log(""HaiHaiHai)       --> this function dosen't change the value of anything outside of it like it is not changing the value or affecting the value of 'a' in this example so this is a pure function
}





Impure function
-->Aisa function jo ki bahar ke value ko badle woh hai impure function.
E.g.:
let a = 12;

function(){
    a++;        -->this function is changing or affecting the value of a in this example so this is an impure function
}



Closures
-> Ek function jo return kare ek aur function aur return hone wala function hamesha use kare parent function ka koi variable

E.g.:
function arpa(){
    let a = 12;
    return function(){
        console.log(a);
    }
}



Lexical Scoping
-> The scope of the variable inside a function is called as lexical scoping.

E.g.:
function arpa(){
    let a = 10;         -->'a' can be used in function arpa, jyra and sada, this is the scope   of 'a'
    function jyra(){
        let b = 9;      -->'b' can be used in function jyra and sada, this is the scope of 'b'
        function sada(){
            let c = 30; -->'c' can be used only in the sada, this is the scope of 'c'
        }
    }
}

Hoisting in expression
E.g.:
