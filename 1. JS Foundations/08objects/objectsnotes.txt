When are objects made ?
Whenever you want to fill the information of specific thing or anything you use objects.

For eg.:->
let obj = {
name: `Aryan`, --> Here (name) is key and (`Aryan`) is value
age: 21,
favfood: `Masala Dosa`
}

let ar = `name`

obj.name --> Output -> Aryan
obj.ar --> Output ->Won't give output to access element like this in reference form [] are used
obj[ar] --> Output -> Aryan (Now you can access with reference form because you used brackets [])

Nesting and Deep-searching
let info = {
name: `Aryan`,
Address: {
country: `India`,
city: `Mumbai`,
Location: {
lat: 10,
lng: 25,
}
}
}

info.Address.Location.lng --> Output -> 25
--> this is how to access the element using deep-searching and this {{}} type is called as nested objects

Destructuring of objects

Taking above example

let{lat,lng} = info.Address.Location;
console.log(lat); --> Output -> 10
console.log(lng); --> Output -> 25

Spread operator in 0bjects
let obj = {
name: `Aryan`,
age: 21,
favfood: `Masala Dosa`
}

let obj2 = {...obj} --> Copies the content of the one object into a new object (...)

let obj3 = Object.assign({}, obj) --> this too copies one object into a new object

let obj4 = Object.assign({dream: Supreme Leader of World}, obj) -> adds dream and it's value in the copied obj
